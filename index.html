04 Train
Systems / Vendors
Local GPUs (primary) ✓ RunPod — SOC 2 Type II ✓ Hugging Face — SOC 2

Standard Secure: Local GPUs primary. RunPod SOC 2 overflow when approved by client. Data may transfer via S3 → RunPod. Instances ephemeral — wiped on termination.
Model Storage
Trained LoRA (safetensor) stored on Hugging Face private repo. 2FA + auth keys. Currently Pro plan; Enterprise Hub available for SOC 2 audit trails + SSO.
RED CODE Protocol

Contract

05 Generate
Systems / Vendors
IPP / Generation Platform ✓ FAL — SOC 2 Type II 3rd-party models (Runway, Kling, etc.) VidMob / 3rd-party insights
Critical Architecture
Client data does NOT pass through APIs. Prompts sent to 3rd-party models via FAL contain no brand assets, source data, or PII. Base result returned → LoRA model applied locally on Kartel infra.
Performance Data
3rd-party creative intelligence data (e.g. VidMob) ingested to inform generation parameters. Lower sensitivity — aggregated performance metrics, not core brand assets. Received via API or secure file transfer from brand's vendor.
Controls
No client data to APIs FAL: private API keys + encrypted comms Full metadata audit trail 2FA gated IPP access
06 Deliver
Systems / Vendors
✓ CIP Platform — AWS ✓ AWS S3 Frame.io Google Drive WeTransfer / Figma
Delivery Controls
Client selects preferred delivery method. CIP recommended for built-in approval workflow and audit trail. All transfers encrypted. Client approval required before final release.
Performance Feedback
Once outputs are distributed, performance data loops back from the brand's media analytics vendor (VidMob, etc.) to the Generate phase — creating a continuous optimization cycle.
Data Destruction
NIST 800-88 secure wipe Written certification to client LoRA model retained (client-owned)
Ownership
100% client-owned IP. Deliverables, LoRA models, and generated content are all client property.

TRUST & COMPLIANCE

Key Takeaways

Key takeaway: Raw client data transforms into a LoRA model at phase 04 — from that point forward, only the model (learned patterns, not source data) and its outputs exist. Third-party APIs only receive prompts — never client data, brand assets, or PII. Performance insights from 3rd-party vendors (e.g. VidMob) are a separate, lower-sensitivity data stream that creates a continuous feedback loop between Deliver and Generate.
